# Quiz API

REST API for managing quiz questions.



## 🚀 Deploy

**Aplicação online:** [https://quiz-api.onrender.com](https://quiz-api.onrender.com)


## Technologies

- Node.js
- Express
- Javascript
- HTML & CSS

## Features

- Create quizzes with questions and answers
- Play quizzes and see their scores
- Manage their quizzes (edit/delete)
- Authenticate via JWT
